# Contributing to this CSV Visualizer

## Mailing List

<a href="https://groups.google.com/g/csv-coreutils-dev" target="_blank">Link to Mailing List</a>

We discuss most of the development on our mailing list
and not in GitHub Issues. If you have an idea, improvement
or want to help out in other ways, make sure to join.

## Other Contributions

Some authors may want to test how much blood, sweat and tears
you are willing to shed for this useless mini-project. 

In that case, you can also just email the authors using the email written
in the top comment above every (Go) source file.
